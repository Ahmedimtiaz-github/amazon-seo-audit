# Amazon SEO Audit Tool

A web-based listing optimization tool that takes Jungle Scout and Helium 10 exports and generates an AI-powered SEO report for any Amazon ASIN — no backend, no installation, runs entirely in the browser.

---

## Live Demo

🔗 **[Try it live →]https://amazon-seo-audit.vercel.app**

---

## What It Does

Upload four data exports, enter a Brand and ASIN, and the tool produces a full SEO audit report in under 30 seconds:

- **Keyword Opportunities Table** — pulled from Helium 10 Cerebro, showing search volume, your current organic rank, competitor average rank, and the gap
- **Current Listing** — title and bullets extracted directly from the uploaded Title & Bullets sheet
- **AI-Optimized Draft** — a new title and 5 bullet points generated by Gemini AI, prioritizing the highest-volume keywords where the ASIN is not ranking or has a large rank gap vs competitors
- **Strategy Explanation** — the AI explains which keywords it prioritized and why
- **One-click Copy** — copy individual fields or the full listing to clipboard

## Files Required

| Upload Slot | File | Source |
|---|---|---|
| JS — All Products | Brand-wide product CSV | Jungle Scout → Products |
| JS — ASIN Data | ASIN-specific product CSV | Jungle Scout → Products (single ASIN export) |
| Helium 10 Cerebro | Keyword research CSV | Helium 10 → Cerebro |
| Title & Bullets | Current listing content | Provided audit sheet (.xlsx) |

---

## How to Use

1. Open the tool in any modern browser
2. Enter the **Brand Name** and **Audit ASIN**
3. Upload the 4 required files (drag & drop supported)
4. Paste a free **Gemini API key** from [aistudio.google.com](https://aistudio.google.com)
5. Click **Generate SEO Report**

---

## Getting a Free API Key

1. Go to [aistudio.google.com](https://aistudio.google.com)
2. Sign in with a Google account
3. Click **"Get API Key"** → **"Create API key"**
4. Copy and paste it into the tool

The Gemini API free tier allows 1,500 requests/day —

## How the AI Prompt Works

The tool sends a structured prompt to Gemini containing:

- Product metrics (price, rating, BSR, category)
- Top 15 keyword opportunities ranked by monthly search volume, each annotated with the ASIN's current organic rank and the competitor average rank
- The existing title and bullet points
- Explicit rules for title format (under 200 chars, keyword-led, sentence-case) and bullet format (ALL-CAPS hook, benefit-focused, 150–250 chars)
- An instruction to prioritize keywords where the ASIN is **not ranking** or has a **large rank gap** vs competitors

The model returns structured JSON (`title`, `bullets`, `reasoning`) which is parsed and rendered directly into the report.
